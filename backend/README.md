# StoryTide - Backend

The API and logic engine for StoryTide, built with Hono and deployed on Cloudflare Workers.

## Key Stack
- **ORM**: Prisma (Accelerate)
- **Engine**: Hono
- **Runtime**: Cloudflare Workers

## Quick Start
```bash
npm install
npm run dev
```

Refer to the [Root README](../README.md) for full setup and deployment instructions.
